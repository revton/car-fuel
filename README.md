## Primeiros Passos (Getting Started)

- Leia `docs/DIRETRIZES.md` para entender arquitetura, padrões de API, qualidade e JDK 17.
- Organize o trabalho com `docs/PROJECTS.md` e crie issues usando `docs/templates/ISSUE_TEMPLATE.md`.
- Estruture PRs empilhadas conforme `docs/STACK-PR.md` para acelerar reviews e merges.
