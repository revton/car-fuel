Aplicação para monitorar e analisar o consumo de combustível do seu carro e melhorar a eficiência.

## Primeiros Passos (Getting Started)

- Leia `docs/DIRETRIZES.md` para entender arquitetura, padrões de API, qualidade e fluxos de PRs.
- Prepare o ambiente de desenvolvimento seguindo `docs/DEVELOPER_SETUP.md`.
- Organize o trabalho com `docs/PROJECTS.md` e `docs/PROCESSO.md`, criando issues com base em `docs/templates/ISSUE_TEMPLATE.md`.
- Para PRs empilhadas, siga o guia `docs/STACK-PR-GHSTACK.md` e, para testes, consulte `docs/TESTING.md`.
